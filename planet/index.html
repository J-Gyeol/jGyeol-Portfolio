PLANET 클론 웹사이트

부동산 브랜드 랜딩 구조를 기반으로 메인 비주얼 캐러셀, 서비스 소개, 비즈니스 섹션, CTA를 반응형으로 구현한 프로젝트입니다.

프로젝트 개요

`Invest in Your Future` 메시지를 중심으로, 프리미엄 부동산 브랜드의 신뢰감과 고급스러운 인상을 전달하는 원페이지 랜딩입니다. 히어로 슬라이드부터 정보 카드, 소개 섹션, 혜택 섹션, CTA까지 자연스럽게 이어지도록 구성했습니다.

기획 의도

첫 화면에서 브랜드 감성을 명확히 전달하고, 아래 섹션에서 투자 가치와 생활 가치를 점진적으로 설득하도록 설계했습니다.

  • 메인 캐러셀로 핵심 카피와 비주얼을 반복 노출해 브랜드 인지 강화
  • 정보 카드와 소개 섹션으로 서비스 방향과 차별 포인트 전달
  • 마지막 CTA 섹션에서 관심 행동으로 이어지는 전환 동선 구성

구현 방식

Bootstrap 5의 그리드, 네비게이션, 오프캔버스, 캐러셀 컴포넌트를 중심으로 구조를 구축했습니다. 모바일 우선 레이아웃을 적용해 화면 크기에 따라 카드 정렬과 섹션 간격이 유연하게 바뀌도록 구현했습니다.

  • 고정 헤더 + 오프캔버스 메뉴로 모바일 탐색 편의성 확보
  • Carousel-fade를 사용해 메인 비주얼 전환을 부드럽게 구성
  • Bootstrap 유틸리티 클래스 활용으로 유지보수성과 생산성 강화

회고

Bootstrap 컴포넌트를 단순 적용하는 데서 끝내지 않고, 카피와 섹션 순서를 조정해 브랜드형 랜딩의 설득 구조를 만드는 데 집중했습니다. 이후 같은 구조를 다른 산업군 랜딩에도 재활용할 수 있도록 클래스와 콘텐츠 블록을 단순화했습니다.
